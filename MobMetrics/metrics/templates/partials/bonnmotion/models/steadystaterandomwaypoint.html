How to use SteadyStateRandomWaypoint
Speed mean (-o) Required

Mean node speed used to define the steady-state speed interval.

  • Units m/s
  • Enter Positive number
Speed delta (-p) Optional

Half-width of the speed interval around the mean. The valid range is 0 ≤ speedDelta < speedMean.

  • Units m/s
  • Enter Number from 0 to less than speed mean
    Resulting interval: [speedMean - speedDelta, speedMean + speedDelta].
    Default: 0.0
Pause mean (-k) Optional

Mean pause time used to define the steady-state pause interval.

  • Units seconds (s)
  • Enter Non-negative number
    Default: 0.0
Pause delta (-l) Optional

Half-width of the pause interval around the mean. The valid range is 0 ≤ pauseDelta ≤ pauseMean.

  • Units seconds (s)
  • Enter Number from 0 to at most pause mean
    Resulting interval: [pauseMean - pauseDelta, pauseMean + pauseDelta].
    Default: 0.0
Example:
-o 1.0 -p 0.5 -k 10 -l 5
