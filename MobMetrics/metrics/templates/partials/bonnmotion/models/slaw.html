How to use SLAW
Number of waypoints (-w) Optional

Total number of waypoints generated for the SLAW process.

  • Enter Integer ≥ 1
    Default: 1000
Minimum pause time (-p) Optional

Minimum pause time between visits to waypoints.

  • Units seconds (s)
  • Enter Non-negative number
    Default: 10
Maximum pause time (-P) Optional

Maximum pause time used by the model's power-law pause generation.

  • Units seconds (s)
  • Enter Positive number
    Default: 50
Levy exponent for pause time (-b) Optional

Beta parameter used in the heavy-tailed pause-time distribution.

  • Enter Positive number
    Default: 1
Hurst parameter (-h) Optional

Controls the self-similarity of waypoint distribution.

  • Enter Typically between 0 and 1
    Default: 0.75
Distance weight (-l) Optional

Weight exponent used when preferring nearer candidate waypoints.

  • Enter Positive number
    Default: 3
Clustering range (-r) Optional

Distance threshold used to form waypoint clusters.

  • Units meters (m) / distance units
  • Enter Positive number
    Default: 50
Cluster ratio (-Q) Optional

Number of clusters selected during the trace-generation phase.

  • Enter Integer ≥ 1
    Default: 5
Waypoint ratio (-W) Optional

Number of waypoints selected from the chosen clusters.

  • Enter Integer ≥ 1
    Default: 5
Example:
-w 1000 -p 10 -P 50 -b 1 -h 0.75 -l 3 -r 50 -Q 5 -W 5
