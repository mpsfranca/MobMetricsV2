How to use Boundless
Time Step (-t)Optional

Simulation update interval (Δt).

  • Units seconds (s)
  • Enter Positive number
    Default: 0.1
Max. Acceleration Change (-m)Optional

Limits how fast speed can increase/decrease per second.

  • Units m/s²
  • Enter Positive number
    Default: 1.5
Alpha (Max. Angular Change) (-s)Optional

Limits how fast the heading can turn.

  • Units rad/s
  • Enter Positive number
    Default: π/2
Example:
-t 0.1 -m 1.5 -s 1.5
