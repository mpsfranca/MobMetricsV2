How to use ManhattanGrid
Speed change probability (-c) Optional

Probability that a node changes its speed after each update distance.

  • Enter Number between 0 and 1
    Default: 0.2
Min. speed (-e) Optional

Minimum speed allowed for nodes.

  • Units m/s
  • Enter Non-negative number
    Default: 0.5
Mean speed (-m) Optional

Average movement speed of nodes on the grid.

  • Units m/s
  • Enter Positive number
    Default: 1.0
Max. pause (-o) Optional

Maximum pause time when a node decides to stop.

  • Units seconds (s)
  • Enter Non-negative number
    Default: 120.0
Pause probability (-p) Optional

Probability that a node pauses after an update step.

  • Enter Number between 0 and 1
    Default: 0.0
Update distance (-q) Optional

Distance interval after which the model may update speed, pause state, or turning behavior.

  • Units meters (m)
  • Enter Positive number
    Default: 5.0
Speed standard deviation (-s) Optional

Standard deviation used when generating new speeds.

  • Units m/s
  • Enter Non-negative number
    Default: 0.2
Turn probability (-t) Optional

Probability that a node turns at an intersection instead of continuing straight.

  • Enter Number between 0 and 1
    Default: 0.5
No. of blocks along x-axis (-u) Optional

Number of horizontal street blocks used to divide the scenario width.

  • Enter Integer ≥ 1
    Default: 10
No. of blocks along y-axis (-v) Optional

Number of vertical street blocks used to divide the scenario height.

  • Enter Integer ≥ 1
    Default: 10
Example:
-c 0.2 -e 0.5 -m 1.0 -o 120 -p 0.0 -q 5.0 -s 0.2 -t 0.5 -u 10 -v 10
