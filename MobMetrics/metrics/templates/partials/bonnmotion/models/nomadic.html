How to use Nomadic
Average nodes per group (-a) Optional

Average number of mobile nodes per group.

  • Enter Positive number
    Default: 3.0
Max distance to group center (-r) Optional

Maximum deviation of each node from its group center or reference point.

  • Units meters (m)
  • Default: 2.5
Group size deviation (-s) Optional

Standard deviation used when sampling the size of each group.

  • Default: 2.0
Reference point max pause (-c) Optional

Maximum pause time of the group reference point.

  • Units seconds (s)
  • Default: 60.0
Example:
-a 3 -r 2.5 -s 2 -c 60
