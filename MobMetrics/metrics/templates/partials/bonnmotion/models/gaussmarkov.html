How to use GaussMarkov
Angle standard deviation (-a) Optional

Controls how much the direction changes from one update to the next.

  • Units radians
  • Enter Non-negative number
    Default: 0.125π
Max. speed (-h) Optional

Upper bound for node speed.

  • Units m/s
  • Enter Positive number
    Default: 1.5
Update frequency (-q) Optional

Time interval for updating speed and direction.

  • Units seconds (s)
  • Enter Positive number
    Default: 2.5
Speed standard deviation (-s) Optional

Controls how much the speed changes between updates.

  • Units m/s
  • Enter Non-negative number
    Default: 0.5
Bounce at boundaries (-b) Optional switch

Keeps nodes inside the simulation area by making them bounce at the boundaries instead of allowing the area to expand.

Min. speed (-m) Optional

Lower bound for node speed.

  • Units m/s
  • Enter Non-negative number
    Default: 0
Gaussian initial speeds (-g) Optional switch

Initializes node speeds with a Gaussian distribution centered around (minspeed + maxspeed)/2

Force uniform speed distribution (-u) Optional switch

Forces speed values to stay within the valid interval using the model's uniform-speed handling.

Example:
-a 0.4 -h 1.5 -q 2.5 -s 0.5 -m 0.0 -b -g -u
