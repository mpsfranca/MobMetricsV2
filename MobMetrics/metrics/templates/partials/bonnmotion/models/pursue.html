How to use Pursue
Max speed (-o) Optional

Upper bound for node movement speed.

  • Units m/s
  • Enter Positive number
    Default: 1.5
Min speed (-p) Optional

Lower bound for node movement speed.

  • Units m/s
  • Enter Positive number
    Default: 0.5
    Constraint: minspeed ≤ maxspeed
Aggressiveness (-k) Optional

Controls how strongly each node moves toward the pursued reference target.

  • Enter Number between 0 and 1
    Default: 0.5
Pursue randomness magnitude (-m) Optional

Controls the strength of the random offset added to pursuit motion.

  • Enter Number between 0 and 1
    Default: 0.5
Example:
-o 1.5 -p 0.5 -k 0.5 -m 0.5
