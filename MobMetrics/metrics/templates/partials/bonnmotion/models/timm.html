How to use TIMM
Path to graph file (-b) Required

Path to the indoor building graph used by the model.

  • Enter Filename or path
    Example: -b building.graph
Max distance per group (-D) Optional

Maximum walking distance for each group. Values are comma-separated, one per group.

  • Units distance units / meters
  • Enter Comma-separated numbers
    Example: -D 10,20,30,40
Group endtime (-E) Optional

End time for each group, given as comma-separated values.

  • Units time units / seconds
  • Enter Comma-separated numbers
    Example: -E 100,200,300,400
Fast speed and variance (-F) Required

Fast movement speed together with its variance, given as two comma-separated values.

  • Units m/s
  • Enter speed,variance
    Example: -F 1.5,0.2
Group sizes (-g) Required

Sizes of all groups, given as comma-separated integers. The sum of these sizes must exactly match the total number of nodes in the scenario.

  • Enter Comma-separated integers
    Example: -g 1,2,3,4
Enable group-one rules (-G) Optional

Enables a special rule set for group one. This is a boolean flag and is disabled by default.

Max graph vertex distance (-m) Optional

Maximum distance allowed between vertices in the building graph.

  • Units distance units / meters
  • Example: -m 5
Minimal group size (-M) Optional

Minimum allowed group size.

  • Example: -M 1
Door opening time and variance (-o) Required

Time needed for door opening and securing, together with variance, given as two comma-separated values.

  • Units time units / seconds
  • Example: -o 3,0.5
Slow speed and variance (-s) Required

Slow movement speed together with variance, given as two comma-separated values.

  • Units m/s
  • Example: -s 1.0,0.1
Group starttime (-S) Optional

Start times for groups, given as comma-separated values.

  • Units time units / seconds
  • Example: -S 0,10,20,30
Example:
-b building.graph -F 1.5,0.2 -g 2,3,4 -o 3,0.5 -s 1.0,0.1 -S 0,10,20
