How to use RandomWalk
Time-limited mode (-t) Choose one mode

Makes each movement segment last for a fixed time interval.

  • Units seconds (s)
  • Enter Positive number
    Use this when you want direction changes to happen after a fixed time span.
Distance-limited mode (-s) Choose one mode

Makes each movement segment last until a fixed distance is covered.

  • Units distance units / meters
  • Enter Positive number
    Use this when you want direction changes to happen after a fixed path length.
Important:

Do not use -t and -s together.

Examples:
-t 60
-s 45
