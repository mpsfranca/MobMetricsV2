How to use RPGM
Average no. of nodes per group (-a) Optional

Average number of mobile nodes assigned to each group.

  • Units nodes
  • Enter Positive number
    Default: 3.0
Group change probability (-c) Optional

Probability that a node changes to another group when it moves into that group's range.

  • Enter Number between 0 and 1
    Default: 0.01
Max. distance to group center (-r) Optional

Maximum deviation allowed between a node and its group reference point.

  • Units meters (m) / distance units
  • Enter Positive number
    Default: 2.5
Group size standard deviation (-s) Optional

Controls how much actual group sizes vary around the average.

  • Units nodes
  • Enter Non-negative number
    Default: 2.0
Example:
-a 3.0 -c 0.01 -r 2.5 -s 2.0
