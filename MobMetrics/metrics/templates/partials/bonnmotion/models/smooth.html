How to use SMOOTH
Range (-g) Required

Communication or interaction range used by the SMOOTH contact calculations.

  • Units meters (m) / distance units
  • Enter Positive integer
Clusters (-h) Required

Number of clusters used during initialization of the scenario.

  • Enter Positive integer
Alpha (-k) Required

Model parameter passed internally as part of the SMOOTH parameter vector.

  • Enter Positive number
Flight length min (-l) Required

Minimum value for the flight-length parameter range.

  • Enter Non-negative integer
Flight length max (-m) Required

Maximum value for the flight-length parameter range.

  • Enter Positive integer
Beta (-o) Required

Model parameter passed internally with alpha and the flight/pause bounds.

  • Enter Positive number
Pause min (-p) Required

Minimum value for the pause-duration parameter range.

  • Units seconds (s)
  • Enter Non-negative integer
Pause max (-q) Required

Maximum value for the pause-duration parameter range.

  • Units seconds (s)
  • Enter Positive integer
Maximum locations (-t) Optional

Sets the upper bound for stored visited locations.

  • Enter Positive integer
    Default: 20000
Example:
-g 50 -h 5 -k 1.5 -l 1 -m 100 -o 1.2 -p 5 -q 30 -t 20000
