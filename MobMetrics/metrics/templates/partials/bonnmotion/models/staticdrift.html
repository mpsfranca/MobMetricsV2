How to use StaticDrift
Input file (-f) Required

File containing the base node positions.

  • Enter Filename or path
    Example: -f positions.txt
    Each non-comment line is parsed as x y or x y z.
Delta X (-X) Optional

Maximum random drift applied around the original x-coordinate. The code samples uniformly from [x - deltaX, x + deltaX].

  • Units distance units / meters
  • Enter Non-negative number
    Default: 0
Delta Y (-Y) Optional

Maximum random drift applied around the original y-coordinate, sampled from [y - deltaY, y + deltaY].

  • Units distance units / meters
  • Enter Non-negative number
    Default: 0
Delta Z (-Z) Optional

Maximum random drift applied around the original z-coordinate in 3D mode only.

  • Units distance units / meters
  • Enter Non-negative number
    Default: 0
Same delta for all axes (-B) Optional shortcut

Convenience flag that sets deltaX, deltaY, and deltaZ to the same value.

  • Enter Non-negative number
    Example: -B 2
    Warning: Combining -B with -X, -Y, or -Z causes random behavior. [file:125]
Recalculation interval (-T) Optional

Recomputes the drifted position every N seconds. If the value is less than or equal to zero, the repeated recalculation is disabled and only one position is used.

  • Units seconds (s)
  • Enter Number
    Default: 0
Example:
-f positions.txt -B 2 -T 5
