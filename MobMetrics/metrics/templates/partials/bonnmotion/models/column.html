How to use Column
Number of groups (-a) Required

Splits nodes into groups for the column formation. The number of nodes must be evenly divisible by the number of groups (e.g., with 50 nodes, each group has 10 nodes.).

  • Units integer
  • Enter Integer ≥ 1
Reference point separation (-r) Optional

Spacing between reference points along the column.

  • Units meters (m)
  • Enter Positive number
    Default: 10.0
Max. distance to group center (-s) Optional

Maximum distance a node may roam around its group reference point.

  • Units meters (m)
  • Enter Positive number
    Default: 2.5
Example:
-a 5 -r 10 -s 2.5
