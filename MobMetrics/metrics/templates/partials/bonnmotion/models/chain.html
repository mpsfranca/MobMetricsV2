How to use ChainScenario
Transition mode (-m) Optional

Selects how nodes transition from one linked scenario to the next.

  • Units integer
  • Enter 0 or 1
    Default: 0
    0: Fast mode; 1: Move mode, where nodes get 2 seconds to reach the new position.
Models to chain Required

One or more scenario model commands to be linked in sequence. ChainScenario instantiates each model, keeps them in order, and then merges all node waypoints into one combined scenario.

  • Enter At least one model command
  • Example model names: RandomWaypoint, GaussMarkov, ManhattanGrid.
Example:
-m 0 "RandomWaypoint -n 50 -x 500 -y 500 -d 300 -h 1.0 -l 0.5 -p 10" "GaussMarkov -n 50 -x 500 -y 500 -d 300 -a 0.75 -h 1.0 -q 0.2"
