How to use TLW
Alpha value (-a) Required

Levy exponent for the flight-length distribution.

  • Enter Number in (0, 2]
    Example: -a 1.5
Beta value (-b) Required

Levy exponent for the pause-time distribution.

  • Enter Number in (0, 2]
    Example: -b 1.2
Maximum simulation size (-s) Required

Upper bound used for both the x and y dimensions of the TLW walk generation.

  • Units meters / distance units
  • Example: -s 1000
Minimum pause time (-m) Optional

Lower bound for accepted pause times.

  • Units seconds (s)
  • Default: 10
Maximum pause time (-M) Optional

Upper bound for accepted pause times.

  • Units seconds (s)
  • Default: 7200
Minimum flight length (-l) Optional

Lower bound for accepted flight lengths.

  • Units meters (m)
  • Default: 5
Maximum flight length (-L) Optional

Upper bound for accepted flight lengths.

  • Units meters (m)
  • Default: 1000
Boundary condition (-c) Required

Controls how the walk behaves at the border of the simulation area.

  • Enter 1 or 2
    Example: -c 1
    1 = wrap-around, 2 = reflection.
Example:
-a 1.5 -b 1.2 -s 1000 -m 10 -M 7200 -l 5 -L 1000 -c 1
