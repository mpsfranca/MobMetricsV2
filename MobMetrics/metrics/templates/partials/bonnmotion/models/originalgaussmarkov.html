How to use OriginalGaussMarkov
Avg speed (-a) Optional

Mean speed used to define the average node velocity.

  • Units m/s
  • Enter Non-negative number
    Default: 0
Max speed (-m) Optional

Upper bound for the node speed.

  • Units m/s
  • Enter Positive number
    Default: infinite
Velocity update frequency (-q) Optional

Time interval between velocity updates.

  • Units seconds (s)
  • Enter Positive number
    Default: 2.5
Velocity standard deviation (-s) Optional

Standard deviation applied to the velocity in each dimension.

  • Units m/s
  • Enter Non-negative number
    Default: 0.1
Alpha (-w) Optional

Controls memory versus randomness in the Gauss-Markov process. (closer to 1 = smoother/more persistent motion; closer to 0 = more random motion.)

  • Enter Number between 0 and 1
    Default: 1.0
Example:
-a 1.0 -m 5.0 -q 2.5 -s 0.1 -w 1.0
